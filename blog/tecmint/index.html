How to Install PostgreSQL on Ubuntu

By John Developer • Updated on November 15, 2025 • Linux Admin

PostgreSQL is one of the world’s most advanced open-source relational databases. In this guide, we will walk you through installing PostgreSQL on Ubuntu, setting up the service, securing your installation, creating a database, and enabling remote access.

1. Update Your System

Before installing PostgreSQL, update your system packages:

sudo apt update && sudo apt upgrade -y

2. Install PostgreSQL

Ubuntu includes PostgreSQL in its default repository. Install it using:

sudo apt install postgresql postgresql-contrib -y

This command installs both PostgreSQL and additional modules such as adminpack.

3. Check PostgreSQL Service Status

After installation, PostgreSQL starts automatically:

sudo systemctl status postgresql

You should see output similar to:

● postgresql.service - PostgreSQL RDBMS
   Loaded: loaded (/lib/systemd/system/postgresql.service; enabled)
   Active: active (exited)

4. Switch to PostgreSQL User

Use the dedicated PostgreSQL user account to access the shell:

sudo -i -u postgres

Enter the PostgreSQL shell:

psql

5. Create a New Database

Inside the PostgreSQL shell, create a database:

CREATE DATABASE myapp;

6. Create a New User and Set a Password

CREATE USER myuser WITH PASSWORD 'strongpassword';

7. Grant Permissions

GRANT ALL PRIVILEGES ON DATABASE myapp TO myuser;

8. Enable Remote Access (Optional)

Edit the PostgreSQL config file:

sudo nano /etc/postgresql/14/main/postgresql.conf

Find and change:

#listen_addresses = 'localhost'

To:

listen_addresses = '*'

Then edit pg_hba.conf to allow connections from specific IPs.

Conclusion

You now have a full PostgreSQL installation on Ubuntu with a database, user, secure configuration, and optional remote access setup. PostgreSQL is a robust and scalable database system perfect for production workloads.
